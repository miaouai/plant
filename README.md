# 🌱 Plant - 二十四节气种植推荐系统

一个基于中国二十四节气的智能植物种植推荐 Web 应用。

## ✨ 功能特性

### 🕐 时间与节气显示
- 实时公历日期时间显示
- 农历日期转换
- 当前节气识别
- 下一节气倒计时

### 🌿 当季种植推荐
- 根据当前节气推荐适合种植的植物
- 详细的植物信息（科属、别名、类型）
- 适宜种植区域说明
- 最佳种植和收获时间
- 养护要点提示

### 📚 二十四节气百科
- 完整的 24 节气卡片展示
- 点击查看详情（气候、物候、习俗）
- 每个节气的特色种植建议
- 中国传统节气文化普及

## 🚀 快速开始

### 🌐 在线访问（推荐）
**GitHub Pages**: https://miaouai.github.io/plant/

### 本地运行
```bash
cd /app/working/mydata/plant
python3 -m http.server 8080
```

然后在浏览器访问：http://localhost:8080

## 📁 项目结构

```
plant/
├── index.html              # 主页面
├── app.js                  # 核心逻辑（农历算法、节气计算、UI 渲染）
├── style.css               # 样式文件
├── data/
│   ├── solar-terms.json    # 24 节气基础数据
│   └── plants.json         # 植物种植推荐数据
├── SKILL.md                # 技能说明文档
└── README.md               # 本文件
```

## 💡 技术亮点

- **纯前端实现**：无需后端服务
- **农历算法**：集成农历转换计算
- **节气引擎**：基于天文算法精确计算
- **响应式设计**：适配手机/平板/电脑
- **数据驱动**：JSON 配置动态生成 UI
